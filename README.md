# Python ile Makine Öğrenmesi ve Derin Öğrenme Çalışmalarım

Bu repo, makine öğrenmesi ve derin öğrenme alanlarında kendimi geliştirmek için izlediğim videolardan ve diğer kaynaklardan uyguladığım proje çalışmalarını içermektedir.  
Her proje, farklı veri setleriyle uygulanmış modelleri, veri ön işleme tekniklerini ve görselleştirmeleri göstermektedir.  

## Projeler

| Proje Adı | Açıklama |
|------------|-----------|
| **KNN.ipynb** | KNN algoritmasıyla diyabet tahmin modeli. Yaş, glikoz, vücut kitle indeksi gibi veriler üzerinden kişinin diyabet hastası olup olmadığını tahmin eder. |
| **IMDB_comment_analysis.ipynb** | Film yorumlarını pozitif veya negatif olarak sınıflandıran doğal dil işleme projesi. |
| **handwriting_recognition.ipynb** | El yazısı ile yazılmış rakamları tanıyan yapay sinir ağı modeli. |
| **multiple_linear_regression.ipynb** | Çoklu doğrusal regresyon modeliyle emlak fiyat tahmini. Alan, oda sayısı ve bina yaşı gibi değişkenleri kullanarak evin fiyatını tahmin eder. |
